# Decreen

Decreen helps teams understand and navigate complex systems by deriving structured maps from existing artifacts (docs, issues, code, shared notes).

This GitHub organization is used for:
- public discussions and product feedback
- the Decreen public roadmap
- open-source connectors and supporting tools (when applicable)

## Public roadmap

We run a public, problem-focused roadmap in **Discussions**.

Use it to:
- upvote problems that reflect real friction in your work
- comment with concrete examples
- add new topics framed as problems rather than solutions
